# 🧠 GenAI Engineer — Free Learning Path

**From Prompt Engineer → GenAI Engineer**
Completely free. Local tools only. No API keys needed.

## What You'll Learn

| Module | Topic | What You'll Build |
|--------|-------|-------------------|
| 0 | Setup | Ollama, Python env, dependencies |
| 1 | Python & NLP Basics | Tokenizers, embeddings, similarity search |
| 2 | Transformers & LLMs | Attention, inference, prompt engineering |
| 3 | LangChain | Chains, memory, agents (fully local) |
| 4 | Vector Databases | ChromaDB, embeddings, vector search |
| 5 | LlamaIndex | Document indexing, query engines, RAG |
| 6 | Advanced RAG | HyDE, multi-query, reranking, evaluation |
| 7 | Fine-tuning | LoRA/QLoRA via Unsloth (Colab), Ollama export |
| 8 | Advanced Topics | Graph RAG, multimodal, evaluation metrics |
| 9 | Interview Prep | 100+ Q&A across all topics |

## Prerequisites

- Python 3.12 (installed — verified)
- Ollama (installed — `ollama.exe` detected)
- VS Code or Jupyter for notebooks
- ~10 GB free disk space for local models

## How to Use This

1. Start with **Module 0** → setup everything
2. Work through **Modules 1→8 sequentially** (each builds on the last)
3. Each `.ipynb` file is self-contained — open in Jupyter/VS Code and run cells
4. Theory files (`.md`) explain the "why" behind the code
5. Finish with **Module 9** interview prep

## Key Principles

- **Zero cost**: everything runs locally or on free Google Colab
- **Learn by building**: every notebook produces something tangible
- **Prompt engineer friendly**: starts from "you already write prompts" → builds the theory underneath
- **Production-ready concepts**: RAG, evaluation, fine-tuning are what companies hire for
